        image_path: Path to receipt image file

================================================================================
                       TROUBLESHOOTING
================================================================================

Tesseract Not Found:
  • Install: brew install tesseract (macOS)
  • Verify: tesseract --version
  • Configure: Set TESSERACT_PATH in .env

Poor OCR Results:
  • Use higher resolution images (1000+ px)
  • Improve lighting when scanning
  • Ensure straight alignment
  • Check image is sharp and in focus

Empty Results:
  • Verify image file is valid
  • Check image preview
  • Try with different image
  • Check Tesseract is working: tesseract --version

For more help, see docs/OCR_MODULE.md troubleshooting section
