## 🎯 THE 6-NODE WORKFLOW

```
                        INPUT
                    (Text or Image)
                          |
                          ▼
        ┌──────────────────────────────┐
        │    NODE 1: OCR NODE          │  → Extract text
        │  (0ms text / 2-5s image)     │
        └──────────────────────────────┘
                          |
                          ▼
        ┌──────────────────────────────┐
        │  NODE 2: EXTRACTION NODE     │  → Parse items
        │       (100-500ms)            │
        └──────────────────────────────┘
                          |
                          ▼
        ┌──────────────────────────────┐
        │  NODE 3: CATEGORIZATION      │  → Assign categories
        │   (50-200ms / 500-2000ms)    │
        └──────────────────────────────┘
                          |
                          ▼
        ┌──────────────────────────────┐
        │    NODE 4: STORAGE NODE      │  → Save to DB
        │       (10-50ms)              │
        └──────────────────────────────┘
                          |
                          ▼
        ┌──────────────────────────────┐
        │    NODE 5: ANALYSIS NODE     │  → Calculate insights
        │     (1-2ms / 500-2000ms)     │
        └──────────────────────────────┘
                          |
                          ▼
        ┌──────────────────────────────┐
        │  NODE 6: RECOMMENDATIONS     │  → Generate tips
        │        (1-500ms)             │
        └──────────────────────────────┘
                          |
                          ▼
                    FINAL RESULTS
        (Insights + Recommendations + DB ID)
```
