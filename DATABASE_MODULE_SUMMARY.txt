💾 DATABASE SCHEMA

   Table: expenses
   ├─ id                INTEGER PRIMARY KEY AUTOINCREMENT
   ├─ date              TEXT (YYYY-MM-DD format)
   ├─ merchant          TEXT (required)
   ├─ category          TEXT (required)
   ├─ amount            REAL (must be > 0)
   ├─ notes             TEXT (optional)
   ├─ created_at        TIMESTAMP (auto)
   └─ updated_at        TIMESTAMP (auto)
