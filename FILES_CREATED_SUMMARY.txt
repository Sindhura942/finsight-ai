================================================================================
                        TROUBLESHOOTING QUICK REFERENCE
================================================================================

"Connection refused"
  → Start API: cd backend && uvicorn main:app --reload

"Port already in use"
  → Use different port: streamlit run streamlit_app.py --server.port 8502

"Module not found"
  → Install dependencies: pip install -r streamlit_requirements.txt

"Dashboard blank"
  → Clear cache: Press R in dashboard

"Data not updating"
  → Reload page: Press F5

"Upload failed"
  → Check file format (JPG/PNG/GIF/BMP) and size (<10MB)

Full troubleshooting guide: See TROUBLESHOOTING.md
